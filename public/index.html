PS04 - Domestic Violence Support and Legal Aid System

A secure, role-based platform connecting survivors of domestic violence with professional counselors and legal advisors. Access confidential support, resources, and legal guidance in a safe environment.

Status 🟢 Live
Backend Render
Frontend Vercel
➜ Access Live Application

Core Features

🔒 Secure Requests

Confidential help requests with end-to-end secure workflows

👥 Role-Based Access

Admin, Survivor, Counsellor, and Legal Advisor roles

💬 Private Chat

Case-based messaging between survivors and professionals

📋 Case Management

Assignment and tracking of survivor queries

Deployment Architecture

  • Frontend: Vercel (this site)
  • Backend: Django on Render (Live API)
  • Database: PostgreSQL on Render
  • Authentication: Django custom user model with roles

Technical Stack

  • Backend: Django 6.0.2
  • Frontend: HTML/CSS/JavaScript (Vercel)
  • Database: PostgreSQL
  • Web Server: Gunicorn
  • Static Files: WhiteNoise
